# CrawlerEngine
  
  這是一個資料競價中心的爬取資料爬蟲框架



| 專案      |功用       | 
| -------- | -------- | 
| CrawlerEngine     | 主程式     | 
| CrawlerEngine.JobWorker     | 執行工作者     | 
| CrawlerEngine.Core     | 爬蟲核心爬取     | 
| CrawlerEngine.Driver     | 瀏覽器模式     |
| CrawlerEngine.Models     | 模型     | 
| CrawlerEngine.Test     | 測試專案     | 

使用方法 
先下載netcore sdk


```
run
dotnet CrawlerEngine.dll
```
